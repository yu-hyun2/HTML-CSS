## 2. HTML 문서 기본구조

**💡 단축키 `! + enter`**

```html
<!DOCTYPE html>
<!--어떤 문서 타입으로 정의되어 있는지 표시(html5를 의미)-->
<html lang="en">
  <!--웹문서의 시작을 나타내는 태그, lang은 언어를 의미-->

  <head>
    <!--웹문서의 문서 정보-->
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
  </head>

  <body>
    <!--실제 보여지는 정보-->
  </body>
</html>
<!--웹문서 끝을 나타내는 태그-->
```
