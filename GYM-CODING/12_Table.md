## ✔️ border-collapse

```css
#movies {
  border-collapse: collapse;
  width: 100%;
}
#movies td,
#movies th {
  border: 1px solid #798bff;
  padding: 8px;
}
#movies th {
  padding-top: 12px;
  padding-bottom: 12px;
  text-align: left;
  background-color: green;
  color: white;
}
#movies tr:nth-child(even) {
  background-color: rgb(155, 155, 155);
}
#movies tr:hover {
  background-color: rgb(201, 201, 201);
}
```
